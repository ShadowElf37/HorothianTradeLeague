About


What is Project Mercury?

Project Mercury is a long-term experiment in using a virtual currency to facilitate trade and fair, organized exaction between people. If you join, you will have access to a number of incredible new features, including an email system, online market, and online place to host requests for help on assignments.

Please give all the input you can about how it works and looks.


How does it work?

Go to the login page through the navigation bar - on the bottom you can sign up. Only certain people are allowed to sign up at this time, and teachers may have to wait a long time before permission in granted via popular concensus. Once logged in, you will have access to a brilliant array of features - test it all out or ask Yovel if you have questions!
